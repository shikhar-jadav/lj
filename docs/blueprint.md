# **App Name**: SoulCanvas

## Core Features:

- Secure Partner Login: Exclusive login for two users with smooth, romantic UI transitions and robust error handling.
- Profile Capture & Display: Utilize browser camera API to capture and save profile pictures to Firebase Storage, displayed in an animated glowing circle.
- Personalized Home Dashboard: Displays profile, floating heart animations, and a daily, AI-generated romantic quote tailored for introspection, refreshed automatically.
- Immersive 3D Shared Gallery: A rotating 3D photo ring using CSS transforms, loaded from Firebase Storage and metadata from Firestore, featuring real-time updates, captions, and favorite highlighting.
- Real-time Collaborative Diary: A shared diary where both users can write, store (in Firestore), and sync entries in real-time, featuring a book-style UI with page flip animations and mood emoji selection.
- Partner Task & Rewards System: Enables users to assign tasks to their partner, track states, accept submissions (text/image to Firestore/Storage), and award points, displayed on clean, animated cards.
- Dynamic Activity Feed & Daily Insights: A combined timeline of photos and diary entries, tracking streaks, and featuring a daily, AI-generated romantic question tool to spark conversation.

## Style Guidelines:

- Primary color: A deep, elegant rose-purple (#CC3399), chosen for its intimate and sophisticated romantic feel against dark backgrounds. Its moderate saturation allows for strong contrast while retaining a premium aesthetic.
- Background color: A subtly hued, very dark charcoal (#1D161B). This low-luminosity, desaturated variant of the primary hue provides a canvas that enhances glowing elements and contributes to a private, premium app experience.
- Accent color: A vibrant purple (#CF67E4), analogous to the primary hue, but brighter and more saturated. This color provides a striking contrast for interactive elements, highlights, and subtle glow effects.
- Headline font: 'Alegreya', a humanist serif, for its elegant, intellectual, and contemporary feel, perfect for capturing emotional weight and premium branding.
- Body font: 'PT Sans', a humanist sans-serif, offering modern aesthetics combined with warmth and excellent readability for longer text blocks such as diary entries.
- Soft, custom-designed icons with subtle glowing effects to align with the romantic, premium theme. Embrace floating hearts and particle animations as primary stylistic elements.
- Mobile-first, clean layouts emphasizing content. Utilize glassmorphism, soft gradients, and shadows to achieve a luxurious feel. Navigation will be intuitive with top bars and/or bottom tabs/slide menus.
- Extremely smooth and expressive animations using Framer Motion. This includes UI transitions, continuous 3D rotations, page flips in the diary, interactive gestures for galleries, and dynamic visual feedback for points and tasks, all enhancing the app's emotional impact and 'premium' feel.
